Installation
============

The most recent release can be installed from `PyPI
<https://pypi.org/project/atomic_femdvr>`_ with uv:

.. code-block:: console

    $ uv pip install atomic_femdvr

or with pip:

.. code-block:: console

    $ python3 -m pip install atomic_femdvr

Installing from git
-------------------

The most recent code and data can be installed directly from GitHub with uv:

.. code-block:: console

    $ uv pip install git+https://github.com//atomic-femdvr.git

or with pip:

.. code-block:: console

    $ python3 -m pip install git+https://github.com//atomic-femdvr.git

Installing for development
--------------------------

To install in development mode with uv:

.. code-block:: console

    $ git clone git+https://github.com//atomic-femdvr.git
    $ cd atomic-femdvr
    $ uv pip install -e .

or with pip:

.. code-block:: console

    $ python3 -m pip install -e .
